Text Alternatives

Provide alternatives for all non-text content

DO

  • Write descriptive alt text for images
  • Include captions for videos
  • Provide transcripts for audio content
  • Use ARIA labels for interactive icons
  • Create text alternatives for complex visualizations

DON'T

  • Use generic alt text (e.g., "image")
  • Include text as part of images
  • Omit descriptions for decorative images
  • Create infographics without text alternatives
  • Use color alone to convey information

Good Alt Text Examples

Chart Image:

"Bar chart showing quarterly revenue growth from Q1-Q4 2024. Q1: $1.2M, Q2: $1.5M, Q3: $1.8M, Q4: $2.2M, representing a 83% annual growth."

Profile Photo:

"Headshot of Sarah Johnson, Marketing Director, smiling and wearing glasses with short dark hair."
